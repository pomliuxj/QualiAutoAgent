# QualiAutoAgent API 自动化测试平台 — 使用指南

## 平台概述

QualiAutoAgent 是一个 AI 驱动的 API 自动化测试平台，支持接口测试用例的自动生成、自动化执行与断言校验、场景测试编排等功能。

## 核心功能

### 1. 单接口测试用例生成
- 基于 Swagger/OpenAPI 文档自动解析接口定义
- 覆盖功能正确性、参数校验、边界值、安全校验、异常容错等多维度
- 自动生成断言规则（HTTP 状态码、响应字段校验）

### 2. 多接口业务场景测试
- 支持多接口串联的业务流程测试
- 自动识别接口间数据依赖关系
- 通过 `$N.field` 语法实现步骤间参数传递
- 支持全局变量动态生成测试数据

### 3. 自动化测试执行
- 批量执行测试用例
- 支持步骤间数据关联（前一步响应作为后一步参数）
- 失败用例自动修复与重跑
- 生成详细的测试报告（Markdown 格式）

### 4. 代码质量分析
- 代码规范检查（命名、格式、注释）
- 潜在 Bug 检测（空指针、边界条件、异常处理）
- 性能问题分析（算法复杂度、资源泄露）
- 安全风险扫描（注入攻击、敏感信息泄露）

### 5. 其他功能
- 全局变量管理（支持 Python 代码动态生成值）
- Host 环境配置（开发/测试/生产环境切换）
- 定时任务与 Cron 调度
- 项目成员管理与权限控制

## 快速开始

### 创建测试用例
1. 在项目中选择"API 接口"页面，添加或导入 API 接口
2. 进入"AI 生成用例"页面
3. 选择"接口用例生成"，输入测试需求
4. AI 自动生成多维测试用例并执行

### 场景测试
1. 选中多个关联接口（如注册→登录→查询）
2. AI 分析接口依赖并设计场景流程
3. 确认场景方案后，AI 按步骤生成并执行场景用例
4. 查看场景测试报告
